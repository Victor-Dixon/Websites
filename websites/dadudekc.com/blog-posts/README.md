# Blog Posts Directory

This directory contains all blog posts for dadudekc.com.

## Directory Structure

```
blog-posts/
├── unprocessed/     # Blog posts that haven't been posted yet
├── processed/       # Blog posts that have been published
└── README.md        # This file
```

## Naming Convention

Blog posts should be named using kebab-case:
- `my-software-engineering-journey.html`
- `learning-wordpress-security.html`
- `building-my-first-game.html`

## File Format

Each blog post should include:
- HTML file with complete blog post
- Optional: Markdown source file (`.md`)
- Optional: Assets folder (images, etc.)

## Workflow

1. **Create** blog post in `unprocessed/` directory
2. **Review** and finalize content
3. **Publish** to website
4. **Move** to `processed/` directory after publishing
