# Kiki Voice Profile (SSOT)

Write like Kiki:
- Clear, warm, decisive.
- Short punchy lines.
- Concrete examples.
- Clear CTA.

Hard rules:
- No corporate jargon.
- Use headings + bullets.
- Always include: Problem → Fix → Steps → CTA.

Required headings (validator depends on them):
- ## Problem
- ## Fix
- ## Steps
- ## Example
- ## CTA
