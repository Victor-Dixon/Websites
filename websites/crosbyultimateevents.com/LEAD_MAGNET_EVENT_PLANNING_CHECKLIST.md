## Your Complete Event Planning Checklist

Planning an event? Use this comprehensive checklist to ensure nothing falls through the cracks. From initial planning to post-event follow-up, we've got you covered.

### Pre-Planning (4-6 Weeks Before)

- [ ] **Define Event Goals**
  - [ ] What is the purpose of this event?
  - [ ] What outcomes do you want to achieve?
  - [ ] Who is your target audience?

- [ ] **Set Budget**
  - [ ] Determine total budget
  - [ ] Allocate funds for: venue, catering, entertainment, decor, marketing
  - [ ] Set aside 10-15% contingency fund

- [ ] **Choose Date & Time**
  - [ ] Check for conflicts (holidays, major events)
  - [ ] Consider guest availability
  - [ ] Have backup date options

- [ ] **Create Guest List**
  - [ ] Initial guest count estimate
  - [ ] Finalize VIP list
  - [ ] Plan for RSVPs and follow-ups

### Venue & Logistics (3-4 Weeks Before)

- [ ] **Venue Selection**
  - [ ] Visit potential venues
  - [ ] Confirm capacity and layout
  - [ ] Review contracts and policies
  - [ ] Book venue and secure deposit

- [ ] **Catering & Bar Service**
  - [ ] Determine menu style (plated, buffet, stations)
  - [ ] Plan bar service (open bar, cash bar, wine/beer only)
  - [ ] Consider dietary restrictions and allergies
  - [ ] **Consider:** Premium mobile bartending service for flexibility

- [ ] **Equipment & Setup**
  - [ ] Audio/visual needs
  - [ ] Tables, chairs, linens
  - [ ] Decor and centerpieces
  - [ ] Lighting requirements

### Food & Beverage (2-3 Weeks Before)

- [ ] **Menu Planning**
  - [ ] Appetizers and hors d'oeuvres
  - [ ] Main course selection
  - [ ] Dessert options
  - [ ] Late-night snacks (if applicable)

- [ ] **Beverage Planning**
  - [ ] Wine selection (red, white, sparkling)
  - [ ] Beer options
  - [ ] Signature cocktails
  - [ ] Non-alcoholic options
  - [ ] Bar service coordination

- [ ] **Service Staff**
  - [ ] Determine number of servers needed
  - [ ] Plan for bartenders
  - [ ] Coordinate service timing

### Entertainment & Activities (2-3 Weeks Before)

- [ ] **Entertainment**
  - [ ] Book DJ, band, or live music
  - [ ] Plan background music
  - [ ] Coordinate sound system

- [ ] **Activities**
  - [ ] Plan interactive elements
  - [ ] Photo booth or photographer
  - [ ] Games or icebreakers (if applicable)

### Final Details (1-2 Weeks Before)

- [ ] **Confirmations**
  - [ ] Finalize guest count
  - [ ] Confirm all vendors
  - [ ] Review contracts one final time
  - [ ] Create day-of timeline

- [ ] **Communication**
  - [ ] Send reminder invitations
  - [ ] Share parking/transportation info
  - [ ] Provide event details to guests

- [ ] **Day-of Preparation**
  - [ ] Create vendor contact list
  - [ ] Prepare emergency contact information
  - [ ] Plan for weather contingencies (if outdoor)
  - [ ] Prepare welcome materials

### Day of Event

- [ ] **Morning Setup**
  - [ ] Vendor arrival coordination
  - [ ] Final venue walkthrough
  - [ ] Confirm all deliveries
  - [ ] Test all equipment

- [ ] **During Event**
  - [ ] Greet guests
  - [ ] Monitor food and beverage service
  - [ ] Ensure timeline stays on track
  - [ ] Address any issues immediately

- [ ] **Event Execution**
  - [ ] Welcome and introductions
  - [ ] Food service timing
  - [ ] Bar service coordination
  - [ ] Entertainment schedule
  - [ ] Closing remarks

### Post-Event (Within 1 Week)

- [ ] **Follow-up**
  - [ ] Send thank-you notes to guests
  - [ ] Thank vendors and staff
  - [ ] Review event success metrics

- [ ] **Evaluation**
  - [ ] What went well?
  - [ ] What would you change?
  - [ ] Gather guest feedback
  - [ ] Document lessons learned
