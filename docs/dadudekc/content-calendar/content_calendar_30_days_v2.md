# content_calendar_30_days_v2 (dadudekc.com)

## Objective
Generate qualified leads for a **2-Week Automation Sprint** (service businesses + founder-led teams **2–20 people**).

## Positioning (one-liner)
**I turn messy small-team ops into simple systems that save 5–15 hours/week (without hiring).**

## Primary customer
- **Service business owners**: events, catering, mobile bartenders, agencies, home services, consultants
- **Founder-led teams**: 2–20 people drowning in follow-ups, scheduling, invoicing, and handoffs
- **Tech-leaning operators** who want clean workflows + dashboards + automation

## Core offer
- **Name**: 2-Week Automation Sprint
- **Promise**: Automate **1–3 workflows end-to-end** + handoff SOP + simple dashboard
- **Price frame**: Cheaper than 1 part-time hire. Pays for itself in hours.

## Lead magnets
- **Automation Audit Checklist** → `/audit`
- **Weekly Scoreboard + Taskboard Template** → `/scoreboard`
- **Client Intake System Template (form → CRM → follow-ups)** → `/intake`

## Channels + cadence rules
### Daily (required)
- **1 Reel/Short (30–60s)** with CTA
- **3–7 Stories** (proof, behind-the-scenes, polls, Q&A)

### Weekly (required)
- **2 carousels** (high-saves) OR **2 LinkedIn text posts**
- **3 blogs** (SEO + credibility)
- **1 email recap**

### Conversion (required)
- **DM script + comment-to-DM magnet**: 3x/week
- **Offer post**: 2x/week (clear deliverables + slots)

## Post structure
### Short (Reels/TikTok)
**Hook (0–2s) → pain → fix → proof/example → CTA**

CTA options:
- DM **“SPRINT”**
- Grab `/audit`
- Grab `/scoreboard`
- Book call

### Carousel
**Swipe 1: promise → 2–7: steps/examples → last: CTA + DM keyword**

### Blog
**problem → costs (time/money) → framework → template → CTA**

### Stories
**poll → 1-liner lesson → behind-the-scenes → CTA sticker**

## DM funnels
### Keyword map
- **SPRINT**: Send sprint overview + ask 3 qualifying questions
- **AUDIT**: Send checklist + ask which workflow is biggest pain
- **INTAKE**: Send intake template + ask what service they sell

### Qualifiers
1) What do you sell?
2) Where do leads come from?
3) What breaks weekly? (scheduling, follow-ups, invoicing, handoffs)
4) How many hours/week lost?
5) Do you want done-for-you or template?

## Weekly theme
- **Week 1**: Pain + diagnosis (make them feel seen)
- **Week 2**: Quick wins + templates (build trust + saves)
- **Week 3**: Proof + builds (show you can ship)
- **Week 4**: Offer + objections + conversion (close)
