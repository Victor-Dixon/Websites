# SEO Tasks Checklist — crosbyultimateevents.com

**Master task log:** “Add SEO tasks for crosbyultimateevents.com (SEO: F)”

---

## P0 (do first)

- **Indexing + sitemap**
  - Verify site is indexable (no sitewide `noindex`)
  - Generate XML sitemap (Yoast/RankMath) and submit in Google Search Console
- **Titles + meta descriptions**
  - Homepage: benefit + location + brand
  - Services pages: targeted, non-duplicative titles
- **H1/H2 structure**
  - 1 H1 per page, clear topical H2s
- **Internal links**
  - Homepage → Services → Booking → Lead magnet
  - Blog posts link to booking + lead magnet
- **Image optimization basics**
  - Descriptive filenames
  - Alt text (service + setting + location when appropriate)

---

## P1 (high leverage)

- **Local SEO**
  - Google Business Profile: categories, services, photos, posts, Q&A
  - Consistent NAP (name/address/phone) across site footer + contact page
- **Schema**
  - `LocalBusiness` (or `FoodEstablishment` if accurate)
  - `Service` schema for key offerings
  - `FAQPage` schema on lead magnet and/or services FAQs
- **Content taxonomy**
  - Blog categories: Event Planning, Menus/Recipes, Client Stories
  - Add tag conventions (Houston, corporate events, private chef, etc.)
- **Core Web Vitals**
  - Lazy-load images
  - Serve WebP
  - Caching plugin (LiteSpeed Cache / WP Rocket)
