## Customization

1. In WordPress admin, go to Screw Tapes â†’ Add New
2. Enter the tape title
3. Add the YouTube Video ID in the sidebar

### Colors & Styling

All colors are defined in CSS custom properties in `/css/style.css`:
- `--primary-color`: Main accent color (default: #ff00ff)
- `--secondary-color`: Secondary accent (default: #00ffff)
- `--accent-color`: Tertiary accent (default: #ffff00)
